Ivanovo Code Jam

Текстильный институт ИВГПУ приглашает всех желающих принять участие в мероприятии Ivanovo Code Jam, которое будет проходить 28 апреля 2013 года в аудитории у230 с 9.00 (вход со стороны ул. Садовая).

Ivanovo Code Jam - это мероприятие, во время которого специалисты из разных областей разработки программного обеспечения (программисты, дизайнеры, менеджеры) могут встретиться и сообща поработать над решением какой-либо проблемы.

Если у вас есть идея, но Вы не знаете как начать её реализовывать, то это мероприятие для Вас! Расскажите о своем проекте, найдите единомышленников и начните его реализацию. Вас ждет интересное общение с разработчиками разного уровня и различных прикладных областей.

Если вы начинающий разработчик, то это мероприятие тоже для Вас. Получите консультацию, примите участие в мозговом штурме или реализации одной из предложенных задач и получите бесценный опыт работы в команде.

Для участия просим вас пройти регистрацию, которая будет доступна в ближайшее время
